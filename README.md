# CodePlants
![](https://github.com/ClubeDosGeeksCoding/CodePlants/blob/master/img/logo.jpg) Arduino â€“ Sensor Reading In Real Time With Socket.io and Jhonny-five; Reading Rain sensors, hygrometer, LM35 in real time and sxibindo in the browser on a local server, a great model to be applied in small plants of irrigation projects.

# Install dependencies
```node
- sudo apt-get install nodejs
- sudo apt-get install npm

- npm install express
- npm install johnny-five
- npm install socket.io
- npm install serialport
```
# Run server
```node
- node app.js
- http://localhost:4000
```
